今天正好是距离微软发布Windows RTM一周年的日子。作为微软押在"后PC时代"的赌注，其桌面模式结合鲜明的Modern UI和触控优化，将原先平板上的概念，拉到了台式机和笔记本身上。继史上最畅销的操作系统——Windows 7之后，微软极力想在Windows 8上证明很多事情。不仅如此，Windows 8还给桌面做出了重大的改变，并且反映了微软自身所发生的许多变化。值此一周年之际，让我们好好回顾一番。







尽管在初期遭受了重大的批评，并且有一大帮的分析师、市场观察人士、甚至媒体人在叫嚣"不合时宜"，"预言"这个新系统"注定要失败"。但Windows 8自推出后便在慢慢地努力。

对于UI上的激进变化、新的应用分布模型(app distribution model)、桌面/开始屏幕的"分裂"等争执似乎逐渐散去，消费者们也逐渐开始采用微软的最新作品。

Windows 8的采用率开始稳步增长，并且在几个月前超过了Vista，现在更是已经占据了5.4%的桌面市场。当然，如果硬要与Windows 7相比的话，这有些微不足道——但对于一个"注定要失败的操作系统"来说，它已经干得很不错了！



Windows 8同样预示了混合设备时代的到来。微软曾试图(并且部分成功地)推动其OEM合作伙伴们，创造创新了设备的外形，而且利用到了Windows 8的"双重特性"。

个别OEM厂商也在它们的新系列产品上获得了成功，但也有不少失意者。不过有一件事是肯定的：带触摸屏的变形设备(convertible devices)仍然"活着"，主要还是因为Windows 8。

尽管第一波变形设备令很多人失望，但软硬件两个方面的进步和质量提升，在未来一定会带来更高的销量。



微软最新的作系统，还为开发者创建和发布应用带来了一种新的设想——Windows Store。开发者可以借此确保和控制其应用的跨设备兼容性，更不用说更简单的赚钱方法了。尽管Windows Store最近突破了10万+应用的大关，理想应用和应用品质的缺失却表明，微软在成功之前还有很多的工作要做。

因此，一年之后，Windows 8已经改变了桌面市场的很多方面，并且"折掉了不少羽毛"。微软给用于带来了对于未来的设想，但其"两面性"和"欠磨练"也令许多人感到了困惑。新UI的"欠解释"，以及不少用户对于新事物的恐惧，都让他们选择了对Win8"保持距离"。

尽管Windows 8的采纳率仍在缓慢地上升，但却没能达到微软此前的预期。更悲哀的是，大多数人对第一波可变形设备的表现"不为所动"。



Windows 8.1带来了许多微小的改动，包括手势和不同尺寸的瓷贴(tiles).

好消息是，微软终于意识到了问题所在，并且在努力修正它们。代号为Blue的Windows 8.1，预计很快就要到达RTM阶段了——改进的UI、更好的控制、以及消费者们所期望的新特性(比如启动直接到桌面)等。

至于Windows 8.1能够重振PC市场的雄风，我们并不抱希望，因为谁也无法读懂市场的预期以及人们究竟想要什么。但无论如何，"打磨"改进后的Windows 8.1，必将愈加发亮。
